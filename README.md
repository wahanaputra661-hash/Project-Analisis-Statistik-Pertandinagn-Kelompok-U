Produk berbasis Microsoft Excel berupa aplikasi analisis statistik pertandingan Final Piala Dunia 2022 yang dirancang untuk menyajikan dan menganalisis data pertandingan final antara Argentina dan Prancis secara sistematis. Aplikasi ini memuat pencatatan data pertandingan, statistik tim dan pemain, susunan pemain, serta kejadian penting selama pertandingan yang diolah secara otomatis dan ditampilkan dalam bentuk grafik serta dashboard sederhana, sehingga memudahkan pengguna dalam memahami jalannya pertandingan, membandingkan performa kedua tim, dan menarik kesimpulan berbasis data secara objektif dan informatif.
